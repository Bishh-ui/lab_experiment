Web Technology Lab Experiments

This repository contains all the practical experiments performed as part of the Web Technology Lab coursework. Each experiment demonstrates different HTML and CSS concepts.

Available Experiments

  • Exercise 1 - Personal Website

    A basic multi-page personal website created using fundamental HTML tags. Includes Home, About, Education, and Contact pages.

  • Exercise 2 - E-Commerce Website

    A static e-commerce website using advanced HTML elements such as tables, forms, figures, details, meter, and progress tags.

  • Exercise 3 - Personal Website with CSS

    Enhanced version of Exercise 1 using Inline, Internal, and External CSS styling techniques.

  • Exercise 4 - E-Commerce Website with CSS

    Enhanced version of Exercise 2 demonstrating the use of Inline, Internal, and External CSS for styling.
