Web Technology Lab Experiments

This repository contains all the practical experiments performed as part of the Web Technology Lab coursework. Each experiment demonstrates different HTML and CSS concepts.

Available Experiments

  • Exercise 1 - Personal Website

    A basic multi-page personal website created using fundamental HTML tags. Includes Home, About, Education, and Contact pages.(Also includes a scientific calculator page of Exp 5 using only the JS code inside the HTML file without linking to the external JS file)

  • Exercise 2 - E-Commerce Website

    A static e-commerce website using advanced HTML elements such as tables, forms, figures, details, meter, and progress tags.

  • Exercise 3 - Personal Website with CSS

    Enhanced version of Exercise 1 using Inline, Internal, and External CSS styling techniques.

  • Exercise 4 - E-Commerce Website with CSS

    Enhanced version of Exercise 2 demonstrating the use of Inline, Internal, and External CSS for styling.

  • Exercise 5 - Calculator with JavaScript

    A fully functional calculator built using JavaScript inside HTML and CSS. Features include basic arithmetic operations and error handling.

  • Exercise 6 - E-Commerce Website with Full Functionality

    A complete e-commerce website with product listings, shopping cart, and user authentication like Create Account and Login features implemented using HTML, CSS, and JavaScript.

  • Exercise 7 - Event Handling in E-Commerce Website and Personal Website

    An interactive e-commerce website demonstrating various JavaScript event handling techniques such as click events, hover events, form submission events, and dropdown change events.Also includes a personal website with event handling features like click, hover, and keyup events to enhance user interaction.
